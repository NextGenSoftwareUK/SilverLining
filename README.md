# Silver Lining

**Find the Positive · OASIS Omniverse**

Silver Lining is an OASIS Omniverse app built on a simple idea: every cloud has one. Users share their challenges and struggles — and the community responds with positive perspectives, practical solutions, and compassionate support. Helping others find their silver lining earns karma.

## What is Silver Lining?

- **Share your challenge** — post what you're going through, anonymously or with your avatar
- **Receive perspectives** — community members respond with reframes, solutions and support
- **Earn karma for helping** — every meaningful response that receives positive feedback awards karma
- **Karma integration** — full OASIS Web4 API integration; karma carries across all apps
- **Safe space** — moderated community with zero tolerance for negativity

## Use Cases

- Mental health support and emotional reframing
- Life advice and practical problem solving
- Career and relationship challenges
- Grief, loss and bereavement support
- Creative blocks and motivation

## Tech Stack

| Layer | Detail |
|---|---|
| Front-end | Single-file `index.html` — inline CSS + vanilla JS |
| Fonts | Orbitron, Rajdhani, Share Tech Mono (Google Fonts) |
| OASIS API | `@oasisomniverse/web4-api@2.0.2` via esm.sh |
| Hosting | Static — any CDN or static host |

## Running Locally

```bash
npx serve .
# or
python -m http.server 8080
```
